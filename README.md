# Preencher-vaga


## Descrição
- software que automatiza o  preenchimento de formularios de empregos

## Documentação

- [Guia de Requisitos](docs/requirements.md)
- [Documentação completa](docs/)

## Instalação
1. Ative venv e `pip install -r requirements.txt`.

## Como Rodar
1. Inicie Chrome com --remote-debugging-port=9222.
2. Navegue a uma página.
3. Rode `python preencher_vaga.py`.
